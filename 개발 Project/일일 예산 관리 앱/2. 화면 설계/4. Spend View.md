## ✅ 목표: 소비 내역 저장 구조

> SpendViewController에서 사용자가 "소비 항목, 금액, 날짜"를 입력하고 저장  
> → CoreData에 저장  
> → HomeViewController에서 날짜별 소비 내역을 참조하여 "오늘 남은 금액" 계산

---

## ✅ CoreData 모델 설계

### 📦 Entity: `SpendingRecord`

|Attribute|Type|설명|
|---|---|---|
|`id`|UUID|고유 식별자 (자동 생성)|
|`title`|String|소비 내용 (예: 커피, 점심 등)|
|`amount`|Integer 32|소비한 금액 (₩ 단위)|
|`date`|Date|소비한 날짜|

---

## ✅ Xcode에서 구성할 때 팁

1. `SpendingRecord`라는 이름의 Entity 생성
    
2. Attribute를 위 표대로 설정
    
3. `Class Definition` 파일을 생성하여 자동 코드 생성
    
4. CoreDataManager에 저장 및 조회 함수 추가

---

## ✅ 예시 CoreDataManager 코드

```swift
func saveSpending(title: String, amount: Int, date: Date) {
    let record = SpendingRecord(context: context)
    record.id = UUID()
    record.title = title
    record.amount = Int32(amount)
    record.date = date

    saveContext()
}

func fetchSpending(on date: Date) -> [SpendingRecord] {
    let request: NSFetchRequest<SpendingRecord> = SpendingRecord.fetchRequest()
    let calendar = Calendar.current
    let startOfDay = calendar.startOfDay(for: date)
    let endOfDay = calendar.date(byAdding: .day, value: 1, to: startOfDay)!

    request.predicate = NSPredicate(format: "date >= %@ AND date < %@", startOfDay as NSDate, endOfDay as NSDate)

    return (try? context.fetch(request)) ?? []
}
```

## ✅ 구현 방식: `UITableView` 사용 추천

|이유|
|---|
|소비 항목은 "제목 + 금액 + 날짜"라는 간단한 구조|
|행별 목록에 적합 (텍스트 기반 데이터)|
|삽입/삭제/수정에 용이함|

## ✅ 구현 예시

### 1. SpendViewController에 UITableView 추가

```swift
private let tableView = UITableView()

override func viewDidLoad() {
    super.viewDidLoad()
    setupUI()
    setupTableView()
    viewModel.fetchTodaySpending()
}
```

---

### 2. ViewModel에서 소비 내역 배열 준비

```swift
final class SpendViewModel {

    var spendingRecords: [SpendingRecord] = []

    func fetchTodaySpending() {
        let today = Calendar.current.startOfDay(for: Date())
        spendingRecords = CoreDataManager.shared.fetchSpending(on: today)
    }
}
```

---

### 3. TableView 구성

```swift
private func setupTableView() {
    tableView.dataSource = self
    tableView.register(UITableViewCell.self, forCellReuseIdentifier: "Cell")
    tableView.translatesAutoresizingMaskIntoConstraints = false
    view.addSubview(tableView)

    NSLayoutConstraint.activate([
        tableView.topAnchor.constraint(equalTo: saveButton.bottomAnchor, constant: 20),
        tableView.leadingAnchor.constraint(equalTo: view.leadingAnchor),
        tableView.trailingAnchor.constraint(equalTo: view.trailingAnchor),
        tableView.bottomAnchor.constraint(equalTo: view.bottomAnchor)
    ])
}
```

---

### 4. UITableViewDataSource 구현

```swift
extension SpendViewController: UITableViewDataSource {
    func tableView(_ tableView: UITableView, numberOfRowsInSection section: Int) -> Int {
        viewModel.spendingRecords.count
    }

    func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell {
        let record = viewModel.spendingRecords[indexPath.row]
        let cell = tableView.dequeueReusableCell(withIdentifier: "Cell", for: indexPath)
        let amount = FormatterUtils.currencyFormatter.string(from: record.amount ?? 0) ?? "₩0"
        cell.textLabel?.text = "\(record.title ?? "") - \(amount)"
        return cell
    }
}
```

---

### 5. 저장 후 테이블 갱신

```swift
viewModel.saveSpending {
    self.viewModel.fetchTodaySpending()
    self.tableView.reloadData()
    self.clearForm()
    self.showAlert("저장 완료", "소비 기록이 저장되었습니다.")
}
```
