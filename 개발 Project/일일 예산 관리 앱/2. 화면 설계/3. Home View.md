## ✅ 앱의 핵심 목적

> **사용자가 오늘 사용할 수 있는 금액을 즉각적으로 확인하고, 지출 흐름을 직관적으로 파악할 수 있도록 돕는다.**

---

## ✅ 홈 화면 정보 우선순위

| 우선순위 | 항목                 | 설명                                   |
| ---- | ------------------ | ------------------------------------ |
| 1️⃣  | **오늘 사용할 수 있는 금액** | 최우선으로 강조 표시 (오늘 예산)                  |
| 2️⃣  | **오늘 예산 계산식 설명**   | 예: `(전일 남은 금액 + 오늘 금액)` 이라는 계산 근거    |
| 3️⃣  | **최근 일주일 요약**      | 사용할 수 있었던 금액, 사용한 금액, 남은 금액 (or 초과분) |
| 4️⃣  | 기타 통계              | 예: 이번 달 누적 지출, 평균 사용액 등 → 추후 확장      |

---

## ✅ 홈 화면 구성안 예시 (UIKit 기준)

### 📱 상단

- 💰 `오늘 쓸 수 있는 금액: ₩32,400` (크게 강조)
    
- 🧾 소제목: `(전일 이월 ₩4,100 + 오늘 예산 ₩28,300)`
    

### 📊 중단

- 최근 7일간 요약 카드
    
    - 일별 잔액/초과/소비량
        
    - 또는 바차트/리스트 뷰로

### 📌 하단

- [소비 기록하기] 버튼 or FAB (Floating Action Button)

## ✅ 탭바 구조 제안

한 화면에 많은 정보를 보여주는 것보다는 **탭바를 활용해 역할을 분리**하면 좋습니다.

| 탭 이름         | 역할                     |
| ------------ | ---------------------- |
| 홈(Home)      | 오늘 예산 표시, 소비 요약        |
| 소비(Spend)    | 소비 기록 입력, 소비 내역 관리     |
| 통계(Stats)    | 주간/월간 리포트, 소비 분석       |
| 설정(Settings) | 월급일, 고정비 수정, 데이터 초기화 등 |
```swift
UITabBarController {
    HomeViewController()
    SpendViewController()
    StatsViewController()
    SettingsViewController()
}
```

## ✅ 디자인 포인트

| 항목             | 제안                                                           |
| -------------- | ------------------------------------------------------------ |
| 💬 "오늘 얼마?" 강조 | `UILabel + Badge`, 혹은 물풍선 효과                                 |
| 🔍 정보 요약       | 작은 `info` 아이콘 or `?` 버튼 클릭 시 계산식/설명 표시                       |
| 📈 최근 일주일 시각화  | `UICollectionView` or `ChartView` 활용 (추후 Charts 라이브러리 연동 가능) |
