### ✅ **[Step 1] 월급과 고정지출 구성**

#### ✔️ 1. `BudgetConfig` (월급/고정지출 설정)

| 속성           | 설명            |
| ------------ | ------------- |
| `salary`     | 월급 총액         |
| `payday`     | 월급일           |
| `fixedCosts` | 고정지출 목록 (관계형) |

---

#### ✔️ 2. `FixedCost` (집세, 보험, 통신비 등)

|속성|설명|
|---|---|
|`title`|항목 이름 (집세, 보험 등)|
|`amount`|월 지출 금액|

🔁 BudgetConfig : FixedCost = 1 : N 관계

---

### ✅ **[Step 2] 하루 예산/소비 데이터**

#### ✔️ 3. `DailyBudget` (일일 예산/소비)

|속성|설명|
|---|---|
|`date`|해당 날짜|
|`availableAmount`|사용할 수 있는 금액 (기본 + 이월)|
|`spentAmount`|당일 실제 사용한 금액|
|`carryOverAmount`|전일 절약해서 이월된 금액|

➡️ 이 테이블은 일자별로 쌓이게 됩니다.

---

### ✅ [Step 3] CoreData 설계 구조

#### 📦 Entity 관계 다이어그램

```less
[BudgetConfig]
 └── 1 : N
     └── [FixedCost]

[DailyBudget]
 └── 독립형
```

---

## ✅ 다음 단계: Xcode CoreData 설계 방식

1. `Resources` 폴더 내 `.xcdatamodeld` 열기
2. Entity 구성:
    - `BudgetConfig`
        - `salary`: Double
        - `payday`: Date
        - Relation → FixedCosts (To-Many)
    - `FixedCost`
        - `title`: String
        - `amount`: Double
        - Relation → BudgetConfig (Inverse)
    - `DailyBudget`
        - `date`: Date
        - `availableAmount`: Double
        - `spentAmount`: Double
        - `carryOverAmount`: Double
