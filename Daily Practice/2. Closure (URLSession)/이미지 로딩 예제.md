## 목표
- URLSession으로 이미지 다운로드
- completion 클로저로 결과 처리
- UIView.animate로 이미지 fade-in

## 핵심 코드
- `loadImage(from:completion:)`: 네트워크 요청 후 클로저 실행
- `showImageWithFadeIn(_:)`: 다운로드 완료 후 애니메이션

## 핵심 개념
- @escaping 클로저 → 비동기 흐름 연결
- [weak self] → 순환참조 방지
- UIView.animate → 뷰 트랜지션

## 학습 포인트
- 실무에서 URLSession은 항상 completion 방식
- 애니메이션 흐름과 데이터 흐름을 분리하는 패턴 구성
