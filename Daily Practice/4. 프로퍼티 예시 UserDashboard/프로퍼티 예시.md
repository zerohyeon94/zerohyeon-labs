# 프로퍼티 실전 예제: UserDashboard

## Stored Property
- 키, 몸무게, 이름 등 실제 데이터 저장

## Computed Property
- BMI, 나이 등 계산된 값 반환

## Lazy Property
- 무거운 뷰는 처음 접근 시 생성

## didSet
- 상태 변화 감지 후 UI 반영

## 실전 포인트
- ViewModel은 struct + computed가 핵심
- ViewController는 class + didSet/lazy 조합이 많음

---
## 🧪 주요 코드 예시

```swift
lazy var settingsView: UIView = {
    print("🌀 SettingsView created")
    let view = UIView()
    // 내부 설정 생략
    return view
}()

var healthScore: Int = 50 {
    didSet {
        updateHealthLabel()
    }
}

var statusSummary: String {
    "\(user.name), \(age)세, BMI: \(user.bmi)"
}
```

