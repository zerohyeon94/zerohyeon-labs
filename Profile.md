## 학습 시스템

- **도구**: Obsidian (Daily Note) + Codex CLI/데스크톱
- **루틴**: 오전 집중 1시간 → 출근 지하철 30-40분 복습 → 평일 근무(10:00-17:00) → 퇴근 지하철 30-40분 정리 → 퇴근 후 운동 전 1시간 산출물 → 운동 후 하루 마무리 → 일요일 주간 회고
- **영어회화 운영**: OPIc/실전 회화를 목표로 `스픽`을 우선 사용한다. 지하철에서는 주제 듣기·표현 3개 저장·작게 따라 말하기, 집에서는 운동 후 10-20분 실제 발화 연습을 기본값으로 둔다.
- **자격증 운영**: 정보처리기사는 평일 핵심 학습을 침범하지 않는 선에서 주 2-3회 선택 과제 또는 주말 단기 목표로 배치한다.
- **모바일 개발 유지**: 영어/자격증을 추가하더라도 iOS/Swift 학습과 모바일 프로젝트 감각 유지를 최우선 축으로 유지한다.
- **Daily 생성 원칙**: 전날 학습 내용을 바탕으로 `Conversations/`에 오늘 학습 파일을 먼저 만들고, Daily Note의 수행 목록 체크박스로도 연결
- **아침 자동화**: Codex 데스크톱 반복 자동화가 매일 오전 6시(KST)에 오늘 Daily Note, `Conversations/` 학습 파일, `Today.md`를 생성하거나 보강
- **저녁 회고**: 하루 정리를 작성한 뒤 "오늘 내용을 피드백해줘"를 수동 요청하여 피드백 품질을 유지
- **간단히 보기**: 맥과 모바일에서 [[Today|오늘 계획]]을 고정 진입점으로 사용
- **운영 노트**: [[Schedule/근무 후 학습 루틴]]
