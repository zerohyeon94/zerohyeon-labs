쉽게 말해 **클래스/구조체/열거형의 멤버 변수**라고 보면 됩니다.

## 1) 프로퍼티의 종류 (Swift)

- **저장 프로퍼티(stored property)**: 메모리에 **실제 값을 저장**

```swift
struct User {
    var name: String          // 저장 프로퍼티
    var age: Int = 0          // 기본값 가능
}
```
    
- **연산 프로퍼티(computed property)**: 값을 **저장하지 않고 계산해서 반환**

```swift
struct Rect {
    var width: Double
    var height: Double
    var area: Double { width * height }  // 연산 프로퍼티 (저장 없음)
}
```
    
- **인스턴스 프로퍼티 vs 타입 프로퍼티**
    
    - 인스턴스: 각 객체가 개별로 가짐
        
    - 타입(=정적) 프로퍼티: `static`/`class` 로 타입 자체가 가짐
        
        ```swift
        struct AppConfig { static let apiBase = "…" }
        ```
        
- **지연 저장 프로퍼티(lazy)**: 처음 접근할 때 초기화

```swift
class VC {
    lazy var cache = DataCache() // 최초 접근 시 생성
}
```
    
- **프로퍼티 옵저버**: 값이 바뀔 때 훅(hook)
    
```swift
var count: Int = 0 {
    willSet { print("-> \(newValue)") }
    didSet  { print("<- \(oldValue)") }
}
```

## 2) ARC와 “프로퍼티로 가진다”의 의미

- **클래스 인스턴스를 프로퍼티에 대입**하면, 그 **소유 객체가 그 인스턴스를 보유(강한 참조, +1)** 한다는 뜻입니다.

```swift
class Owner {
    var child: Child?   // 기본은 strong 보유 (+1)
}
owner.child = Child()   // 여기서 Child의 RC가 +1
owner.child = nil       // 여기서 Child의 RC가 -1 (다른 참조 없으면 deinit)
```
    
- 강한 순환 참조를 피하려면:
    
    - **weak**: 참조는 하되 **RC를 늘리지 않음**(옵셔널이어야 함)
        
    - **unowned**: RC를 늘리지 않음 + **절대 nil 아님**을 가정(해제 후 접근하면 크래시)

```swift
class Child {
    weak var delegate: Parent?   // 순환 차단에 흔히 사용
}
```
    
- **연산 프로퍼티는 저장하지 않으므로** ARC에 직접 영향이 없어요(다만 getter에서 다른 인스턴스를 생성/반환하면 그 순간의 소유 규칙은 여전히 적용).

## 3) 값 타입 안의 “참조 보관” 주의

- `struct`(값 타입)라도 **프로퍼티로 클래스(참조 타입)를 가지면** 그 프로퍼티에 **대입하는 순간 +1** 입니다.

```swift
class Service {}
struct Container {
    var service: Service   // 대입하면 service의 RC가 +1
}
```
    
- `Array`, `Dictionary` 같은 컬렉션에 **클래스 인스턴스를 넣어도 컬렉션이 strong 보유**합니다(추가 시 +1, 제거 시 -1).

## 4) 한눈 정리 (면접/실무 포인트)

- 프로퍼티 = 타입이 **소유하는 상태(멤버 변수)**
    
- 저장/연산, 인스턴스/타입, lazy, 옵저버 등으로 구분
    
- **클래스 인스턴스를 프로퍼티에 저장**하면 기본은 **strong(+1)**
    
- 순환 참조 위험 지점은 **weak/unowned**로 끊기
    
- 연산 프로퍼티는 저장 X → **ARC에 직접 영향 없음**
    
- 값 타입이라도 **참조 타입을 프로퍼티로 가지면 +1** 규칙 동일
