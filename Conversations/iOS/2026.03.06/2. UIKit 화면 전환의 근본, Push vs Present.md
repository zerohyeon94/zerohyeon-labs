**🗣️ 가볍게 시작하기 (Simple)**

UIKit에서 화면을 넘기는 방식은 아주 명확한 두 개의 메서드 이름으로 나뉩니다.

1. **Push (책장 넘기기):** `UINavigationController`라는 관리자가 화면들을 **스택(Stack, 배열)**에 차곡차곡 쌓는 방식입니다.
    
2. **Present (포스트잇 덮기):** 현재 화면(`UIViewController`) 위에 새로운 화면을 **모달(Modal)** 형태로 띄우는 방식입니다.

---

**🤿 딥하게 들어가기 (Deep: 실전 코드와 속성)**

UIKit 코드를 보면 "아, SwiftUI가 이걸 이렇게 줄여준 거구나!" 하고 무릎을 탁 치게 되실 겁니다.

**상황 A: 지출 내역 리스트 ➔ 상세 화면 (Navigation Push)**

- **조건:** 이 방식을 쓰려면 첫 화면이 반드시 `UINavigationController` 안에 담겨서 시작되어야 합니다. (이게 SwiftUI의 `NavigationStack` 역할입니다.)
    
- **코드 흐름:**
    
    ```Swift
    // 1. 다음 화면 객체 만들기
    let detailVC = DetailViewController()
    detailVC.item = selectedItem // 데이터 전달
    
    // 2. NavigationController를 통해 밀어 넣기 (Push)
    self.navigationController?.pushViewController(detailVC, animated: true)
    ```
    
- **뒤로 가기:** 좌측 상단에 기본 백버튼이 생기며, 코드로 빼내고 싶다면 `self.navigationController?.popViewController(animated: true)`를 호출합니다.
    

**상황 B: 타이머 시작 화면 ➔ 집중 타이머 뷰 (Modal Present)**

- **조건:** 네비게이션 컨트롤러 유무와 상관없이, 어떤 `UIViewController`에서든 바로 띄울 수 있습니다.
    
- **코드 흐름:**
    
    ```Swift
    // 1. 띄울 화면 객체 만들기
    let timerVC = FocusTimerViewController()
    
    // 2. 띄우는 스타일 지정 (이 부분이 핵심!)
    // .fullScreen은 화면 전체를 덮는 SwiftUI의 fullScreenCover와 동일
    // .pageSheet는 살짝 아래로 내려가 있는 SwiftUI의 sheet와 동일 (iOS 13+ 기본값)
    timerVC.modalPresentationStyle = .fullScreen 
    
    // 3. 현재 화면이 직접 새로운 화면을 띄움 (Present)
    self.present(timerVC, animated: true, completion: nil)
    ```
    
- **뒤로 가기(닫기):** 이 화면은 스와이프로 안 닫히므로(fullScreen일 경우), 타이머 뷰 내부에 닫기 버튼을 만들고 `self.dismiss(animated: true)`를 호출해서 스스로를 모달에서 내려야 합니다.

---

### 🚨 UIKit만의 골칫거리: "누가 누구를 띄웠지?"

SwiftUI는 상태(State) 기반이라 `isPresented = true`만 하면 시스템이 알아서 띄워주지만, UIKit은 **명령형**입니다.

그래서 화면 B 위에 화면 C를 띄우고(Present), 또 화면 D를 띄운(Present) 상태에서, 갑자기 맨 처음 화면 A로 한 번에 돌아가고 싶다면? `dismiss`를 연달아 호출하거나, 최상위 부모를 찾아서 닫아주는 등 **'뷰 컨트롤러 계층(Hierarchy)'** 관리를 개발자가 직접 꼼꼼하게 해줘야 합니다. (이래서 이전에 나눴던 수다에서 Coordinator 패턴이 등장했던 것이죠!)

---

**💡 오늘의 요약**

- **SwiftUI `NavigationStack`** = **UIKit `UINavigationController` (Push/Pop)**
    
- **SwiftUI `.fullScreenCover` / `.sheet`** = **UIKit `.present` / `.dismiss`**
