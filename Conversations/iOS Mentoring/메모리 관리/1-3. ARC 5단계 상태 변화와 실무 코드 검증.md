# 1-3. ARC 5단계 상태 변화와 실무 코드 검증 (답변)

> 질문: ARC의 5단계 상태 변화를 실제 Swift 코드로 확인할 수 있나요?

---

## 💻 1. 코드 기반 상태 추적 (Alive ~ Deiniting)

객체의 초기 상태와 파괴 시작점은 `init`과 `deinit`을 통해 아주 명확하게 확인할 수 있습니다.

```swift
class MemoryInspector {
    let id: String
    init(id: String) {
        self.id = id
        print("✅ [1. Alive] \(id) 객체가 생성되었습니다.")
    }
    
    deinit {
        print("⏳ [2. Deiniting] \(id) 객체의 deinit이 시작되었습니다. 아직 프로퍼티(\(id))에 접근 가능합니다.")
    }
}

func testLifecycle() {
    var instance: MemoryInspector? = MemoryInspector(id: "TestObj") // Count 1
    instance = nil // Count 0 -> deinit 호출
}

testLifecycle()
```

## 🔍 2. 눈에 보이지 않는 단계들 (Deinitialized ~ Freed)

`deinit`이 종료된 후의 단계들은 Swift 코드 내에서 직접 관찰하기 어렵습니다. 왜냐하면 **객체 본체가 이미 유효하지 않기 때문**입니다. 하지만 `weak` 참조를 통해 그 흔적을 추적할 수 있습니다.

### A. Deinitialized & Freed 상태 확인법
이 단계들은 객체 본체의 데이터는 날아갔지만 **Side Table은 살아있는** 상태입니다.

```swift
var weakRef: MemoryInspector?

func verifyZombieState() {
    let strongInstance = MemoryInspector(id: "ZombieTester")
    weakRef = strongInstance
    
    print("--- Scope 종료 전 ---")
} // strongInstance 해제 시점

verifyZombieState()

// 여기서 weakRef를 확인하면?
if weakRef == nil {
    print("💀 [4. Freed -> 5. Dead 사이] 본체는 사라졌고, Side Table을 통해 nil임이 확인되었습니다.")
}
```

## 🛠️ 3. 실무 도구: Memory Graph Debugger

코드로 출력하는 것보다 더 정확하게 상태를 보는 방법은 Xcode의 **Memory Graph Debugger**를 활용하는 것입니다.

1.  **캡처 버튼**: Xcode 하단 디버그 바에서 '세 개의 원이 연결된 아이콘'을 클릭합니다.
2.  **객체 선택**: 좌측 사이드바에서 특정 클래스를 선택합니다.
3.  **참조 확인**: 
    - 실선(Strong): 객체를 살려두고 있는 상태 (**Alive**)
    - 점선(Weak): 객체의 사망 여부를 감시 중인 상태 (**Side Table 관여 중**)

## ⚡ 4. 팁: `isKnownUniquelyReferenced`

실무에서 "이 객체가 나만 참조하고 있는(Alive이며 Count=1) 상태인가?"를 확인하여 불필요한 복사를 막고 싶을 때 사용합니다. (주로 CoW 구현 시 사용)

```swift
var obj = MemoryInspector(id: "Uniqueness")
// 이 함수는 오직 클래스 타입의 참조 횟수가 1인지를 Bool로 반환합니다.
print(isKnownUniquelyReferenced(&obj)) // true
```

---

## 💡 요약
*   **Alive/Deiniting**: `init`/`deinit` 로그로 확인 가능.
*   **Deinitialized/Freed**: `weak` 변수가 `nil`이 되는 시점과 메모리 디버거의 점선(Side Table)으로 유추 가능.
*   **Dead**: 메모리 디버거에서 해당 주소 자체가 완전히 사라진 상태.
