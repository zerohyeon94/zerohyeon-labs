**🗣️ 가볍게 시작하기 (Simple)**

"서버 통신 없이 핸드폰이 스스로 생각하게 만든다." 이게 CoreML의 핵심입니다.

- **장점 1 (속도):** 서버 갔다 오는 시간(Latency)이 0초입니다. 실시간 영상 처리에 필수죠.
    
- **장점 2 (프라이버시):** 사용자의 얼굴이나 생체 데이터를 서버로 보낼 필요가 없습니다. (Apple이 제일 좋아하는 마케팅 포인트!)
    
- **장점 3 (비용):** 비싼 GPU 서버를 안 써도 됩니다. 사용자의 아이폰이 알아서 계산해 주니까요.

---

**🤿 딥하게 들어가기 (Deep: PyTorch ➔ CoreML 변환 파이프라인)**

사용자님이 PyTorch로 만든 모델(`.pt` 또는 `.pth`)은 아이폰이 바로 이해할 수 없습니다. 아이폰은 **`.mlpackage` (또는 `.mlmodel`)**라는 전용 포맷만 알아듣거든요.

이 통역사가 바로 **`coremltools`**라는 파이썬 라이브러리입니다.

**1단계: 파이썬에서 변환하기 (Mac이나 Colab에서 실행)**

```Python
import torch
import coremltools as ct

# 1. 학습된 PyTorch 모델 로드
model = MyFocusModel()
model.load_state_dict(torch.load("focus_model.pth"))
model.eval() # 평가 모드로 전환 필수!

# 2. 더미 데이터(입력 모양 알려주기) 생성
# 예: 눈 깜빡임(1) + 머리 움직임(1) = 입력 2개
dummy_input = torch.rand(1, 2) 

# 3. 변환 (Trace 방식)
traced_model = torch.jit.trace(model, dummy_input)
mlmodel = ct.convert(
    traced_model,
    inputs=[ct.TensorType(name="input_data", shape=dummy_input.shape)]
)

# 4. 저장
mlmodel.save("FocusSenseModel.mlpackage")
```

**2단계: Xcode에 넣기 (Drag & Drop)** 방금 만든 `FocusSenseModel.mlpackage` 파일을 Xcode 프로젝트에 끌어다 놓습니다. 그러면 Xcode가 자동으로 **Swift 클래스**를 만들어줍니다! (마법 같죠? ✨)

**3단계: Swift에서 추론하기 (Inference)**

```Swift
// 자동으로 생성된 클래스 사용
let model = try! FocusSenseModel(configuration: MLModelConfiguration())

// 입력 데이터 준비 (MLMultiArray)
let input = try! MLMultiArray(shape: [1, 2], dataType: .float32)
input[0] = 0.5 // 눈 깜빡임
input[1] = 0.1 // 머리 움직임

// 예측(Prediction) 
// 서버 통신 없이 폰 내부에서 0.001초 만에 결과가 나옵니다.
if let output = try? model.prediction(input_data: input) {
    print("현재 집중도 점수: \(output.score)") 
}
```

**🧠 하드웨어 이야기: ANE (Apple Neural Engine)**

펌웨어 경험이 있으시니 이 부분이 재밌으실 겁니다. 아이폰 칩(A시리즈, M시리즈) 안에는 CPU, GPU 말고 **NPU(Neural Processing Unit)**가 따로 들어있습니다. 애플은 이걸 **ANE (Apple Neural Engine)**라고 부릅니다.

- CoreML을 쓰면, OS가 알아서 판단합니다.
    
    - "이건 단순 행렬 연산이네? **ANE**로 보내서 배터리 아끼면서 처리하자."
        
    - "이건 그래픽 처리가 많네? **GPU**로 보내자."
        
    - "이건 복잡한 분기문이네? **CPU**가 처리해."
        

개발자가 따로 설정할 필요 없이, CoreML 모델을 쓰기만 하면 이 강력한 하드웨어 가속을 공짜로 누릴 수 있습니다.
