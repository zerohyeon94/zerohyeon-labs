**🗣️ 가볍게 시작하기 (Simple): 책장 넘기기 vs 포스트잇 붙이기**

앱에서 화면을 이동하는 방식은 크게 두 가지 감성으로 나뉩니다.

1. **Navigation (Push): "책장 넘기기"**
    
    - 오른쪽에서 왼쪽으로 스르륵 화면이 밀려 들어옵니다.
        
    - "내가 어디서부터 들어왔지?"라는 **계층 구조(Hierarchy)**가 중요할 때 씁니다. 기본적으로 좌측 상단에 '뒤로 가기(<)' 버튼이 자동으로 생깁니다.
        
2. **Modal (Present): "새로운 종이(포스트잇) 덮어씌우기"**
    
    - 아래에서 위로 화면이 불쑥 솟아오릅니다. (`.sheet`는 절반만, `fullScreenCover`는 전체를 덮음)
        
    - 이전 흐름과 완전히 분리된 **새로운 작업(Context Switch)**을 시작할 때 씁니다. 유저가 이 작업을 끝내거나(완료) 취소(X 버튼)하기 전까지는 이전 화면으로 돌아갈 수 없게 강제하는 효과가 있습니다.

---

**🤿 딥하게 들어가기 (Deep: 실전 아키텍처 설계)**

새로운 화면을 짤 때마다 "둘 중 뭘 써야 하지?" 고민되신다면, 만들고 계신 앱의 성격에 대입해 보면 아주 명확해집니다.

**상황 A: 가계부 앱의 지출 내역 보기 ➔ `NavigationStack`** 지출 내역 리스트에서 특정 결제 건(예: 스타벅스 커피)을 눌러 상세 내역으로 들어가는 상황입니다.

- **이유:** '전체 리스트 ➔ 특정 항목'이라는 명확한 깊이(Depth)가 있습니다. 상세 내역을 보고 나서 자연스럽게 뒤로 가기를 눌러 다시 리스트로 돌아와야 하므로 `Navigation`이 제격입니다.

```Swift
NavigationStack {
    List(transactions) { item in
        // 누르면 DetailView로 밀고 들어감
        NavigationLink(destination: DetailView(item: item)) {
            Text(item.title)
        }
    }
    .navigationTitle("지출 내역")
}
```

**상황 B: 집중력 앱의 타이머 시작 ➔ `fullScreenCover`** 메인 화면에서 세팅을 마치고 드디어 '집중 모드 시작' 버튼을 누르는 상황입니다.

- **이유:** 타이머가 돌아가는 동안에는 다른 화면(설정, 내역 등)으로 쉽게 빠져나가면 안 됩니다. 화면 전체를 꽉 채워서 유저를 현재 작업에 '가둬두는' 형태가 필요하므로 `fullScreenCover`가 완벽합니다.

```Swift
struct MainView: View {
    @State private var isTimerRunning = false
    
    var body: some View {
        Button("집중 모드 시작") {
            isTimerRunning = true // 상태를 true로 바꾸면 화면이 덮임
        }
        // 아래에서 위로 화면 전체를 덮어버림
        .fullScreenCover(isPresented: $isTimerRunning) {
            FocusTimerView(isPresented: $isTimerRunning)
        }
    }
}
```

### 🚨 iOS 개발자라면 챙겨야 할 디테일 (Dismiss)

- `NavigationStack`에서는 유저가 스와이프하거나 기본 '<' 버튼을 누르면 알아서 이전 화면으로 돌아갑니다.
    
- 하지만 `fullScreenCover`로 띄운 타이머 화면은 **스스로 내려가는 기능이 없습니다.** 반드시 띄워진 화면(자식 뷰) 내부에서 종료 버튼을 만들고, `@Environment(\.dismiss)`를 호출하거나 부모로부터 받은 `@Binding` 값을 `false`로 바꿔서 명시적으로 창을 닫아주어야 합니다.

```Swift
// FocusTimerView 내부의 종료 버튼 액션
@Environment(\.dismiss) private var dismiss

Button("집중 종료") {
    dismiss() // 화면을 아래로 내려보냄!
}
```
