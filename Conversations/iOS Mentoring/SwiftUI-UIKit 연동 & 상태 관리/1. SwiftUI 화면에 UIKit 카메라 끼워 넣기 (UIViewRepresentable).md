**🗣️ 가볍게 시작하기 (Simple)**

SwiftUI로 UI를 그리면 코드가 정말 짧아지고 직관적이어서 재미있죠. 그런데 방금 16번 수다에서 만든 카메라 화면(`AVCaptureVideoPreviewLayer`)을 띄우려고 하면 막막해집니다. 왜냐하면 **SwiftUI에는 아직 완벽한 네이티브 카메라 프리뷰 컴포넌트가 없거든요.**

결국 카메라나 지도 같은 복잡한 요소들은 기존의 UIKit을 가져와서 써야 합니다. 이때 UIKit 뷰를 SwiftUI 세계로 초대하기 위해 씌우는 '마법의 포장지'가 바로 **`UIViewRepresentable`**입니다.

---

**🤿 딥하게 들어가기 (Deep: 포장지의 두 가지 핵심 원리)**

`UIViewRepresentable` 프로토콜을 채택하면, 시스템이 우리에게 딱 두 가지 필수 질문을 던집니다.

**1. `makeUIView` (탄생의 순간)**

- **의미:** "SwiftUI가 처음 이 화면을 그릴 때, 어떤 UIKit 뷰를 만들어줄까?"
    
- **역할:** 여기서 `UIView`를 생성하고, 카메라 세션을 연결하고, 기본 세팅을 마친 뒤 리턴합니다. 앱 생명주기 동안 **딱 한 번만** 호출됩니다.
    

**2. `updateUIView` (변화의 순간)**

- **의미:** "SwiftUI 쪽의 상태(State)가 변했는데, 그에 맞춰서 이 UIKit 뷰도 업데이트해 줄래?"
    
- **역할:** SwiftUI의 특성상 상태가 변하면 뷰를 새로 그리지만, UIKit 뷰는 새로 만들어지지 않고 이 메서드만 호출됩니다. 여기서 전달받은 새로운 데이터를 UIKit 뷰에 덮어씌워 줍니다.

**💻 실전 코드 스케치**

FocusSense 앱의 백그라운드에 집중도를 측정하는 카메라를 띄운다고 상상해 보겠습니다.



```Swift
import SwiftUI
import AVFoundation

// UIKit의 카메라 뷰를 SwiftUI용으로 포장!
struct CameraPreviewView: UIViewRepresentable {
    let captureSession: AVCaptureSession
    
    // 1. 처음 한 번만 실행됨
    func makeUIView(context: Context) -> UIView {
        let view = UIView(frame: UIScreen.main.bounds)
        
        let previewLayer = AVCaptureVideoPreviewLayer(session: captureSession)
        previewLayer.frame = view.bounds
        previewLayer.videoGravity = .resizeAspectFill
        
        view.layer.addSublayer(previewLayer)
        return view
    }
    
    // 2. SwiftUI 상태가 변할 때마다 실행됨 (여기선 카메라 세션만 유지하면 되니 비워둠)
    func updateUIView(_ uiView: UIView, context: Context) {
        // UI 업데이트 로직
    }
}
```

### 🎨 SwiftUI의 진가: 오버레이(Overlay) 얹기

포장된 카메라 뷰를 만들었다면, 이제 SwiftUI의 진짜 매력을 발휘할 차례입니다. `ZStack`이나 `.overlay`를 사용해서 카메라 뷰 위에 SwiftUI로 만든 세련된 타이머 게이지나 UI 요소들을 겹쳐 올리는 거죠.

```Swift
struct TimerView: View {
    var body: some View {
        ZStack {
            // 1. 맨 밑에 방금 만든 UIKit 카메라 뷰를 깝니다.
            CameraPreviewView(captureSession: mySession)
                .ignoresSafeArea()
            
            // 2. 그 위에 SwiftUI로 만든 세련된 UI를 덮어씌웁니다.
            VStack {
                Text("현재 집중도: 95%")
                    .font(.largeTitle)
                    .foregroundStyle(.white)
                
                Spacer()
                
                // 집중 모드 해제 화면을 띄우는 버튼 등
                Button("종료") { ... }
            }
        }
    }
}
```

이렇게 하면 **무거운 로직(카메라, AI 추론)은 UIKit과 백그라운드에 맡기고, 가볍고 예쁜 UI 처리는 SwiftUI로** 가져가는 아주 세련된 구조가 완성됩니다.
