**🗣️ 가볍게 시작하기 (Simple): "포스트잇과 대형 철제 캐비닛"**

- **UserDefaults (포스트잇):** '다크 모드 켜짐', '자동 로그인 여부' 같은 아주 간단하고 가벼운 설정값을 저장할 때 씁니다. 모니터에 잠깐 붙여두는 포스트잇 같은 역할이죠.
    
- **SwiftData (철제 캐비닛):** '2026년 3월 5일 집중 시간 60분', '3월 6일 집중 시간 45분'처럼 계속해서 쌓이는 방대한 기록들을 날짜별로 정렬하고 검색해야 할 때 쓰는 강력한 로컬 데이터베이스입니다. (과거에 쓰던 CoreData의 최신형 버전입니다.)

---

**🤿 딥하게 들어가기 (Deep: SwiftData를 움직이는 3가지 핵심 톱니바퀴)**

SwiftData는 딱 3가지만 기억하면 끝납니다. 이전에 TCA나 DI 컨테이너를 배우셨던 감각이라면 정말 쉽게 느껴지실 겁니다!

1. **`@Model` (서류 양식):** 어떤 데이터를 저장할지 규격을 정하는 곳입니다.
    
2. **`ModelContainer` (캐비닛 본체):** 실제 데이터가 안전하게 저장되는 하드디스크 공간입니다.
    
3. **`ModelContext` (캐비닛 관리자):** 화면(View)에서 데이터를 새로 넣거나 뺄 때 부르는 매니저입니다.

---

**💻 실전 코드: 집중 기록 데이터베이스 만들기**

**Step 1: 서류 양식 만들기 (`@Model`)** 과거에는 복잡한 설정 파일이 필요했지만, 이제는 클래스 위에 `@Model`이라는 단어 하나만 붙이면 그 즉시 데이터베이스 테이블이 완성됩니다!

```Swift
import SwiftData
import Foundation

// 이 클래스 전체가 하나의 데이터베이스 표(Table)가 됩니다.
@Model
class FocusRecord {
    var id: UUID
    var focusDate: Date      // 언제 집중했는지
    var durationMinutes: Int // 몇 분이나 집중했는지
    var finalScore: Int      // 최종 점수

    init(focusDate: Date, durationMinutes: Int, finalScore: Int) {
        self.id = UUID()
        self.focusDate = focusDate
        self.durationMinutes = durationMinutes
        self.finalScore = finalScore
    }
}
```

**Step 2: 앱 껍데기에 캐비닛 설치하기 (`ModelContainer`)** 우리가 이전 수다에서 뇌(Store)를 주입했던 그 앱의 시작점에, 이 철제 캐비닛도 같이 설치해 줍니다.

```Swift
import SwiftUI
import SwiftData

@main
struct FocusSenseApp: App {
    var body: some Scene {
        WindowGroup {
            FocusTimerView()
        }
        // "이 앱은 FocusRecord라는 서류를 보관할 캐비닛을 쓸 거야!" 라고 선언
        .modelContainer(for: FocusRecord.self) 
    }
}
```

**Step 3: 화면에서 관리자 부르기 (`@Query`와 `ModelContext`)** 이제 통계 화면(HistoryView)에서 저장된 데이터를 불러오거나, 새로운 타이머 기록을 추가해 보겠습니다.

```Swift
import SwiftUI
import SwiftData

struct HistoryView: View {
    // 1. 관리자(Context) 부르기: 데이터를 추가/삭제할 때 씁니다.
    @Environment(\.modelContext) private var context
    
    // 2. ⭐️ 캐비닛 뒤지기 (@Query): 저장된 모든 기록을 가져와서 날짜순으로 정렬까지 해줍니다!
    @Query(sort: \FocusRecord.focusDate, order: .reverse) 
    private var records: [FocusRecord]

    var body: some View {
        NavigationStack {
            List(records) { record in
                VStack(alignment: .leading) {
                    Text("\(record.durationMinutes)분 집중!")
                        .font(.headline)
                    Text("점수: \(record.finalScore)점")
                        .font(.subheadline)
                }
            }
            .navigationTitle("내 집중 기록")
            .toolbar {
                Button("임시 기록 추가") {
                    addDummyRecord()
                }
            }
        }
    }
    
    // 새로운 기록을 캐비닛에 넣는 함수
    private func addDummyRecord() {
        let newRecord = FocusRecord(
            focusDate: Date(), 
            durationMinutes: 60, 
            finalScore: 95
        )
        // 관리자에게 새 서류를 캐비닛에 넣어달라고 지시!
        context.insert(newRecord) 
    }
}
```
