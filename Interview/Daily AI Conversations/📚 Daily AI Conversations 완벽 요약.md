## **1️⃣ AI의 기본 정리 (2026.04.13)**

### 📌 AI, ML, DL의 포함 관계
- **AI** (가장 넓음): 기계가 인간의 지능을 모방하는 모든 기술
- **ML**: 데이터를 통해 스스로 규칙을 학습하는 AI의 하위 분야
- **DL**: 신경망을 깊게 쌓아 복잡한 패턴을 스스로 학습하는 ML의 하위 분야

### 📌 머신러닝의 3가지 학습 방법
- **지도학습**: 문제와 정답을 함께 제공 (분류, 회귀)
- **비지도학습**: 정답 없이 데이터의 숨은 패턴 발견 (군집화, 차원 축소)
- **강화학습**: 보상을 최대화하며 학습 (알파고, 자율주행)

### 📌 분류 vs 회귀
| 구분 | 분류 (Classification) | 회귀 (Regression) |
|------|--------|--------|
| **결과값** | 범주형 (이산적) | 수치형 (연속적) |
| **예시** | 이미지 인식, 졸음 판정 | 집값 예측, 주가 예측 |

### 📌 모델 평가의 핵심 개념
- **과소적합(Underfitting)**: 모델이 덜 학습됨
- **과적합(Overfitting)**: 모델이 훈련 데이터만 외움
- **데이터 분할**: Train/Validation/Test 분리
- **평가 지표**: 정확도, 정밀도, 재현율, F1-Score

## **2️⃣ 딥러닝의 핵심 원리 (2026.04.14)**

### 📌 신경망의 진화 과정
1. **단층 퍼셉트론**: 직선으로 데이터 분리만 가능 → **XOR 문제 해결 불가**
2. **다층 퍼셉트론**: 입력-은닉-출력층으로 확장 → 비선형 분류 가능
3. **딥러닝**: 은닉층 2개 이상으로 복잡한 패턴 학습

### 📌 딥러닝의 3가지 마법
- **활성화 함수**:
  - Sigmoid: 확률 해석이 쉬우나 기울기 소실 문제
  - **ReLU**: 현재 표준, 기울기 소실 해결
  - Softmax: 다중 분류의 확률 계산
  
- **손실 함수 + 경사하강법**: 오차를 최소화하는 가중치 찾기
- **오차 역전파**: 출력층에서 입력층으로 오차 역방향 전달

### 📌 CNN vs RNN
| 모델 | 특화 데이터 | 핵심 | 활용 |
|------|--------|------|------|
| **CNN** | 이미지/영상 | 공간적 특징 추출 | 얼굴 인식, 졸음 감지 |
| **RNN** | 텍스트/음성 | 시계열 정보 기억 | 자동 번역, 챗봇 |

## **3️⃣ ZipJoong 프로젝트 분석 (2026.04.20)**

### 📌 프로젝트 개요
- **목표**: 스마트폰에서 실시간 집중도(졸음/이석) 감지
- **모델**: MobileNetV3-Small (CNN 기반)
- **학습 방식**: Multi-Task Learning (얼굴 존재 + 졸음 동시 판단)

### 📌 왜 이 선택들을 했는가?
1. **CNN 선택**: 이미지의 공간적 특징(눈 감김, 얼굴 형태) 추출
2. **MobileNetV3-Small**: 모바일의 속도/배터리 극적 최적화
3. **Multi-Task Learning**: 1회 추론으로 2가지 결과 (연산 효율 극대화)
4. **PyTorch + CoreML**: 최신 표준 프레임워크 + iOS 변환 최적화

### 📌 실전 문제 해결 (도메인 변화)
**문제**: Kaggle(차량 운전자)로 학습한 모델이 책상 환경에서 오분류
**해결**: Vision Framework (70%) + CoreML (30%) 하이브리드
- 수학적으로 명확한 EAR(눈 깜빡임 비율) 메인
- 패턴 인식 부족분을 딥러닝으로 보조

### 📌 모델 개선 전략
1. **지식 증류**: 큰 모델의 노하우를 작은 모델에 전수
2. **고도화된 데이터 증강**: 모니터 불빛, 그림자 추가로 도메인 차이 극복
3. **하드 네거티브 마이닝**: 모델이 가장 헷갈리는 케이스 집중 공략
4. **파인 튜닝**: Pre-trained 가중치 활용으로 적은 데이터로도 높은 성능
