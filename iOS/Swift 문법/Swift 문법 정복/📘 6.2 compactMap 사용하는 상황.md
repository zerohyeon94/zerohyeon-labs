## ✅ 1. 옵셔널 배열에서 `nil` 제거할 때

```swift
let input = ["1", "2", "three", "4"]
let numbers = input.compactMap { Int($0) }
print(numbers) // [1, 2, 4]
```

- `map`은 모든 값을 그대로 유지하면서 변형
    
- `compactMap`은 변환 후 `nil`인 값은 제거

👉 문자열 배열을 숫자로 바꾸되, 실패한(`nil`) 값은 제거하고 싶을 때 사용

## ✅ 2. JSON 디코딩 후 nil이 섞인 데이터를 정리할 때

```swift
struct User: Decodable {
    let name: String?
}

let users: [User] = ...
let validNames = users.compactMap { $0.name }
```

- `name`이 `nil`인 유저는 제외하고, **유효한 name만 추출**

---

## ✅ 3. 옵셔널 프로퍼티가 있는 모델에서 리스트 필터링할 때

```swift
class Product {
    var price: Double?
    init(price: Double?) { self.price = price }
}

let list = [Product(price: 3.0), Product(price: nil), Product(price: 7.5)]
let prices = list.compactMap { $0.price }  // [3.0, 7.5]
```

- `nil`인 price는 제외하고 실제 값만 추출

---

## ✅ 4. Enum에서 `.some` 값만 추출할 때

```swift
enum Result<T> {
    case success(T)
    case failure
}

let results: [Result<Int>] = [.success(1), .failure, .success(5)]
let values = results.compactMap {
    if case .success(let val) = $0 {
        return val
    } else {
        return nil
    }
}
// values → [1, 5]
```

- 패턴 매칭으로 일부 케이스만 뽑아내기

## ✅ 5. RxSwift에서도 자주 사용됨 (예: `Driver` → 옵셔널 스트림 필터링)

```swift
myTextField.rx.text
    .compactMap { $0 } // nil 제거
    .bind(to: viewModel.inputText)
    .disposed(by: disposeBag)
```

- 옵셔널인 `.text`를 안전하게 처리
