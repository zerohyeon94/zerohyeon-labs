## 📌 예시 다시 정리

```swift
func showAlert(with message: String) {
    print("🔔 \(message)")
}
```

### ✅ 호출 방법:

```swift
showAlert(with: "알람 메시지")
```

---

## 🧠 외부 파라미터 이름의 목적

|호출 형태|느낌|
|---|---|
|`showAlert("문구")`|❌ 무엇을 전달하는지 불명확|
|`showAlert(with: "문구")`|✅ "무엇을 통해 보여준다"는 의도가 드러남|

이런 명확한 문법은 **Apple의 API 설계 철학**에서도 중요한 포인트입니다.  
예를 들어:

```swift
UIView.animate(withDuration: 0.3)
```

## 🎯 함수 파라미터 이름 정리

```swift
func 함수이름(외부이름 내부이름: 타입)
```

예:

```swift
func greet(to name: String) {
    print("Hello, \(name)")
}
greet(to: "Yeonghyeon")
```

---

## 💡 외부 이름 생략하려면?

```swift
func greet(_ name: String) {
    print("Hi, \(name)")
}
greet("Yeonghyeon")  // ✅ 외부이름 생략
```
