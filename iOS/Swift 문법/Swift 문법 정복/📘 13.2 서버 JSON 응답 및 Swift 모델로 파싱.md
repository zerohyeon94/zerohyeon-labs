## 🧾 1. Swift 모델 정의 (Codable)

```swift
struct User: Codable {
    let id: Int
    let name: String
    let skills: [String] // ✅ Array
}
```

---

## 🌐 2. URLSession으로 JSON 가져오기

```swift
func fetchUsers(completion: @escaping ([User]) -> Void) {
    guard let url = URL(string: "https://example.com/api/users") else { return }

    URLSession.shared.dataTask(with: url) { data, response, error in
        guard let data = data else { return }

        do {
            let users = try JSONDecoder().decode([User].self, from: data) // ✅ [Dictionary] → [User]
            DispatchQueue.main.async {
                completion(users)
            }
        } catch {
            print("❌ 디코딩 실패: \(error)")
        }
    }.resume()
}
```

- 서버 응답 `[Dictionary]` → `[User]` 배열로 변환
    
- `skills`는 내부적으로 `[String]` 배열로 자동 매핑됨

## 👀 3. ViewController에서 출력

```swift
fetchUsers { users in
    for user in users {
        print("🧑‍💻 \(user.name)의 스킬:")
        for skill in user.skills {
            print("- \(skill)")
        }
    }
}
```

## 📌 여기서 사용된 컬렉션 구조 요약

| 항목      | 타입                                | 사용 이유                  |
| ------- | --------------------------------- | ---------------------- |
| 전체 응답   | `[User]` (Array)                  | 유저 리스트를 순서 있게 보여주기 위해  |
| skills  | `[String]` (Array)                | 유저마다 가진 기술 스킬을 나열하기 위해 |
| JSON 원본 | `[[String: Any]]` (Dictionary 배열) | JSON 자체는 딕셔너리 배열 형태    |
