## 📘 1단계 Day 3: 제어문 (조건 & 반복문)

### 🔹 목표

- 조건 분기(`if`, `guard`, `switch`)와 반복문(`for`, `while`) 사용 방법 익히기
- `guard`의 사용 이유 이해

---

### 🔹 조건문 정리

| 문법       | 설명                    | 예시                                      |
| -------- | --------------------- | --------------------------------------- |
| `if`     | 조건 만족 시 실행            | `if age > 20 { ... }`                   |
| `guard`  | 조건 불만족 시 early return | `guard age > 20 else { return }`        |
| `switch` | 다중 조건 분기              | `switch direction { case "left": ... }` |

> 💡 `guard`는 **조건을 만족하지 않으면 빠르게 종료**하는 경우에 사용합니다. `early exit` 방식으로 코드 가독성이 좋아집니다.

---

### 🔹 반복문 정리

| 문법             | 설명                  | 예시                           |
| -------------- | ------------------- | ---------------------------- |
| `for in`       | 범위 또는 컬렉션 반복        | `for i in 0..<5 {}`          |
| `while`        | 조건이 참인 동안 반복        | `while count > 0 {}`         |
| `repeat while` | 최소 1회 실행 후 반복 조건 검사 | `repeat { ... } while x > 0` |

---

### 🔹 예제 코드

```swift
let score = 85

// if
if score >= 90 {
    print("A")
} else if score >= 80 {
    print("B")
} else {
    print("C or below")
}

// guard
func checkAge(_ age: Int?) {
    guard let age = age, age >= 20 else {
        print("입장 불가")
        return
    }
    print("환영합니다! \(age)세")
}

checkAge(25)
checkAge(nil)

// switch
let direction = "left"
switch direction {
case "left":
    print("←")
case "right":
    print("→")
default:
    print("unknown")
}

// for in
for i in 1...5 {
    print("🔥 \(i)회차 반복")
}

// while
var count = 3
while count > 0 {
    print("카운트 다운: \(count)")
    count -= 1
}
```

---

### 🧠 시니어 개발자 꿀팁

- ✅ `guard`는 **ViewController에서 validate 함수 구현** 시 깔끔하게 처리할 수 있는 필수 문법입니다.
- ✅ `switch` 문에 **default는 항상 명시**하세요. 미래에 case 추가를 놓치면 문제가 됩니다.
- ✅ `forEach`와 혼동하지 않도록 `for-in`과 클로저 반복은 구분해서 사용하세요.
