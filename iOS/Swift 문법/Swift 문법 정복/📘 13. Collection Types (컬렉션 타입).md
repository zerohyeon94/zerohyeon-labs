Swift의 기본 컬렉션은 아래 3가지입니다:

|타입|설명|
|---|---|
|`Array`|순서가 있는 값의 목록|
|`Set`|순서가 없고, 중복되지 않는 값의 집합|
|`Dictionary`|키-값 쌍으로 구성된 컬렉션|

---
### 1. **Array**

```swift
var fruits: [String] = ["Apple", "Banana", "Cherry"]

fruits.append("Durian")
print(fruits[0]) // "Apple"
```

- **순서가 보장됨**
    
- index 기반 접근 가능

---

### 2. **Set**

```swift
var numbers: Set<Int> = [1, 2, 3, 1, 2]

print(numbers) // [1, 2, 3] - 중복 제거됨
```

- **순서 없음**, **중복 허용 안 함**
    
- 빠른 탐색, 합집합·교집합에 유리

---

### 3. **Dictionary**

```swift
var user: [String: String] = [
    "name": "YeongHyeon",
    "role": "iOS Developer"
]

print(user["name"] ?? "") // "YeongHyeon"
```

- **키-값 쌍 구조**
    
- 키는 `Set`처럼 고유해야 함
---
## 🎯 요약

|컬렉션|특징|유용한 상황|
|---|---|---|
|Array|순서 유지, 중복 허용|리스트, 순회|
|Set|중복 없음, 순서 없음|중복 제거, 빠른 탐색|
|Dictionary|키-값 구조|사용자 정보 저장, JSON 파싱
