## 🔧 고차 함수란?

> 함수를 **매개변수로 받거나**, **함수를 반환하는 함수**

Swift에서는 컬렉션(`Array`, `Dictionary`, `Set`)에 대해 많이 사용됩니다.

---

## 1️⃣ `map` – 변환

> 배열의 각 요소를 다른 값으로 변환하여 **새 배열 생성**

```swift
let numbers = [1, 2, 3]
let doubled = numbers.map { $0 * 2 }
// 결과: [2, 4, 6]
```

---

## 2️⃣ `filter` – 조건 필터링

> 조건에 **부합하는 요소만 남김**

```swift
let numbers = [1, 2, 3, 4, 5]
let even = numbers.filter { $0 % 2 == 0 }
// 결과: [2, 4]
```

---

## 3️⃣ `reduce` – 누적 계산

> 배열을 단일 값으로 줄여감

```swift
let numbers = [1, 2, 3, 4]
let sum = numbers.reduce(0) { $0 + $1 }
// 결과: 10
```

---

## ✍️ 클로저 표현 간소화

```swift
numbers.map { $0 * 2 }               // 간결한 표현
numbers.map { number in number * 2 } // 명시적
numbers.map({ $0 * 2 })              // 괄호 클로저
```

---

## 🧠 시니어 꿀팁

|팁|설명|
|---|---|
|`filter { !$0.isEmpty }`|빈 문자열, 빈 배열 제거|
|`reduce(into:)`|성능 최적화된 누적|
|`compactMap`|nil 제거하며 변환 (`map + filter(nil)`)|
|`forEach`|단순 반복 실행만 → return, break 불가 주의|
