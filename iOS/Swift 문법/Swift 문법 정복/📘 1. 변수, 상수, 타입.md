## 📘 1단계 Day 1: 변수, 상수, 타입

### 🔹 목표

- `let`, `var` 차이 이해
- 타입 추론과 명시적 선언 구분
- 기본 타입 사용
---
### 🔹 핵심 개념

| 키워드   | 설명               | 예제                           |
| ----- | ---------------- | ---------------------------- |
| `let` | 상수(변경 불가)        | `let age = 30`               |
| `var` | 변수(변경 가능)        | `var name = "Jo"`            |
| 타입 추론 | Swift가 타입을 자동 판단 | `let score = 95 // Int로 추론`  |
| 타입 명시 | 타입을 명확히 지정       | `let height: Double = 180.5` |

> 💡 `let`을 기본으로 사용하고, 변경 필요할 때 `var`로 바꾸는 습관이 좋습니다.

---

### 🔹 예제 코드

```swift
import Foundation

let name = "Yeonghyeon"      // 타입 추론: String
var age = 28                 // 타입 추론: Int

let height: Double = 174.2   // 명시적 선언

print("이름: \(name), 나이: \(age), 키: \(height)")

age += 1
print("내년 나이: \(age)")

// name = "Other"  // ❌ 오류: let은 변경 불가
```
