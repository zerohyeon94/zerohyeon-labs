## 📌 `compactMap`이란?

> `compactMap`은 **옵셔널을 반환하는 클로저**를 실행하고, 그 중 **nil이 아닌 값만 모아서 새로운 배열로 반환**하는 고차 함수입니다.

---

## 🔸 차이: `map` vs `compactMap`

|함수|설명|예시 결과|
|---|---|---|
|`map`|변환만 수행|`[1, nil, 3]` 유지|
|`compactMap`|변환 + nil 제거|`[1, 3]` (nil 제거됨)|

---

## ✅ 예제 1: 문자열 → Int로 변환

```swift
let values = ["1", "two", "3", "four"]

let numbers = values.compactMap { Int($0) }
print(numbers) // [1, 3]
```

- `Int($0)`는 실패 시 nil을 반환
- `compactMap`이 nil 값을 제거한 배열을 반환

---

## ✅ 예제 2: 옵셔널 배열에서 nil 제거

```swift
let maybeNames: [String?] = ["Jo", nil, "Kim"]

let names = maybeNames.compactMap { $0 }
print(names) // ["Jo", "Kim"]
```

---

## ✅ 예제 3: JSON 디코딩 후 nil 필터링

```swift
let rawData = ["1", "2", "a", "3"]

let validIDs = rawData.compactMap { Int($0) }
// ["1", "2", "3"]만 유효한 숫자로 변환됨
```

---

## 🧠 시니어 꿀팁

|팁|설명|
|---|---|
|`compactMap`은 `map + filter { $0 != nil }`의 축약||
|옵셔널 배열 다룰 때 **nil 제거용으로 가장 많이 사용**||
|`compactMapValues`는 딕셔너리에서 nil 제거할 때 사용됨||
|RxSwift에도 `compactMap` 오퍼레이터 있음 (`.compactMap { $0 }`)|
