## 🔧 함수 기본 구조

```swift
func 함수이름(매개변수: 타입) -> 반환타입 {
    // 실행 코드
    return 값
}
```

### 예시

```swift
func greet(name: String) -> String {
    return "Hello, \(name)!"
}

let message = greet(name: "Yeonghyeon")
print(message) // Hello, Yeonghyeon!
```

---

## 🧩 주요 문법 포인트

|개념|예시|설명|
|---|---|---|
|매개변수 기본값|`func greet(name: String = "Guest")`|인자 생략 가능|
|외부 파라미터 이름|`func greet(to name: String)`|`greet(to: "Jo")`|
|반환 없는 함수|`func log()` 또는 `-> Void`|반환 필요 없을 때 사용|
|다중 파라미터|`func sum(a: Int, b: Int)`|쉼표로 나열|
|다중 반환 (튜플)|`-> (Int, String)`|복수 값 반환 가능|

---

## 🧪 예제

```swift
func sum(_ a: Int, _ b: Int) -> Int {
    return a + b
}

func greet(to name: String, isMorning: Bool = true) -> String {
    let time = isMorning ? "Good morning" : "Good evening"
    return "\(time), \(name)!"
}

func userInfo() -> (name: String, age: Int) {
    return ("Yeonghyeon", 28)
}
```

## 🧠 시니어 꿀팁

- ✅ 외부 파라미터를 활용하면 **함수 사용 의도가 더 명확해짐**
    
    ```swift
	func showAlert(message: String)  // ❌
	func showAlert(with message: String)  // ✅
	```
    
- ✅ 기본값(default value)은 **오버로딩을 줄이고** 가독성 향상
    
- ✅ 튜플 반환은 작고 명확한 다중 값을 리턴할 때 적합
    
    - 큰 구조는 별도 모델(struct)로 분리하는 게 좋음
