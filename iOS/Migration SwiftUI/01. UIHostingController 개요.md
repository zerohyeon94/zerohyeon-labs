## 1. 정의 
UIKit 프레임워크 내에서 SwiftUI 뷰를 표시할 수 있도록 해주는 **중재자(Adapter)** 클래스입니다. 
## 2. 역할 
- **브릿지 역할**: 겉모습은 `UIViewController`이지만 내부 콘텐츠는 SwiftUI `View`를 담습니다. 
- **상태 관리**: SwiftUI의 데이터 변경을 UIKit 환경에서도 반영할 수 있게 돕습니다. 
## 3. 기본 코드 구조

```swift
let swiftUIView = StatsView() 
let hostingController = UIHostingController(rootView: swiftUIView)
```
