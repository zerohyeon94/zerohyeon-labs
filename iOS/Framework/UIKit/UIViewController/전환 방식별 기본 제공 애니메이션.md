## 1. **전환 방식별 기본 애니메이션(동작 흐름)**

### (1) **NavigationController (Push/Pop)**

- **Push**: 
	- 새 화면이 **오른쪽에서 왼쪽**으로 밀려 들어옴
- **Pop**: 
	- 현재 화면이 **왼쪽으로 밀려나감** (이전 화면이 보임)
- **의미:** 
	- "계단식" 흐름, 깊이 있게 들어가거나 뒤로 나갈 때 자연스럽게 사용

### (2) **Modal (Present/Dismiss)**

- **Present**:
    - **아래에서 위로**(iOS 13~ 기본: 시트/카드 스타일)
    - 전체화면 modal(이전 버전이나, fullScreen): **아래에서 위로 전체화면 덮기**
- **Dismiss**:
    - **위에서 아래로** 사라짐
- **의미:**
    - 별도의 흐름, 임시로 어떤 작업(설정, 작성, 선택 등) 수행 후 닫는 용도

### (3) **TabBarController**

- **탭 선택 시**
    - **애니메이션 없음** (화면 즉시 전환)
- **의미:**
    - 서로 독립적인 메인 섹션을 이동 (홈, 검색, 마이페이지 등)

### (4) **Custom Transition**

- **직접 구현**
    - 원하는 방향/효과로 전환(페이드, 줌, 플립 등)

## 2. **상황에 맞는 화면 전환 방식 추천**

### (1) **Navigation Push/Pop이 어울리는 상황**

- 정보 구조가 계단식(상세 → 더 상세)
- "뒤로가기"가 자연스러워야 하는 흐름
- 예)
    - 목록 → 상세보기
    - 게시글 리스트 → 게시글 상세 → 댓글 보기

### (2) **Modal Present가 어울리는 상황**

- 본 흐름과는 별개로 "임시 작업"
- 필수 정보가 아닌 선택, 설정, 입력 등
- 예)
    - 회원가입, 로그인
    - 필터/정렬 설정
    - 사진 선택, 알림 안내, 팝업

### (3) **TabBar가 어울리는 상황**

- 동등한 계층의 여러 기능(메인 섹션)
- 서로 다른 업무영역, 독립적인 흐름
- 예)
    - 쇼핑 앱: 홈/카테고리/장바구니/내정보
    - 소셜 앱: 피드/메시지/프로필

### (4) **ChildViewController(컨테이너)**

- 화면 일부에 동적으로 여러 뷰를 넣고 빼야 할 때
- 커스텀 탭, 페이지뷰, 슬라이딩 메뉴 등

### (5) **Custom Transition**

- 특별한 효과가 필요할 때
- ex) 카드 스타일, 페이드, 확대/축소, 슬라이드 등
- (보통 UIKit의 Delegate 패턴으로 직접 커스텀)

## 정리

| 방식                   | 기본 애니메이션/동작                | 어울리는 상황                           |
|------------------------|------------------------------------|----------------------------------------|
| Navigation (Push/Pop)  | 오른쪽→왼쪽(들어옴)/왼쪽→오른쪽(나감) | 상세화면, 단계별 이동, 뒤로가기 필요      |
| Modal (Present)        | 아래→위(나타남), 위→아래(사라짐)      | 임시 작업, 설정, 팝업, 입력              |
| TabBar                 | 애니메이션 없음 (즉시전환)           | 메인 기능 간 이동(독립섹션)              |
| ChildVC/컨테이너       | 개발자 구현에 따라 다름               | 커스텀 레이아웃, 동적 영역               |
| Custom Transition      | 원하는 효과(슬라이드, 페이드 등)       | 특수효과 필요시, UX 강조                |

## 상황별 권장 전환

- 상세보기, 단계별 흐름: **Push**
- 팝업, 임시 설정: **Present(Modal)**
- 앱의 주요 기능 전환: **TabBar**
- 화면의 일부만 동적으로 변경: **ChildVC**
- 특별한 애니메이션 필요: **Custom Transition**
