`sceneDidEnterBackground(_:)`는 **앱이 백그라운드로 전환될 때 호출되는 메서드**로,  
여기서는 앱이 화면에서 사라지는 동안에도 **마무리 작업이나 상태 저장 등을 수행**할 수 있습니다.

---

## ✅ 기본 시그니처

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    // 앱이 백그라운드로 전환될 때 호출됨
}
```

---

## 🔧 여기에 들어갈 수 있는 대표적인 코드 예시

### ✅ 1. Core Data 저장

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    (UIApplication.shared.delegate as? AppDelegate)?.saveContext()
}
```

- `saveContext()`는 Core Data의 변경사항을 영구 저장소에 커밋하는 메서드
    
- 사용자가 앱을 닫더라도 데이터가 보존되도록 함

---

### ✅ 2. UserDefaults 저장

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    let defaults = UserDefaults.standard
    defaults.set(Date(), forKey: "lastUsedDate")
}
```

- 마지막 사용 시간을 저장해두고, 다음 실행 시 복원에 활용할 수 있음

---

### ✅ 3. 네트워크 요청 취소

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    NetworkManager.shared.cancelAllRequests()
}
```

- 앱이 백그라운드에 있을 때 불필요한 네트워크 자원을 아끼기 위해 요청 중단

---

### ✅ 4. 타이머 중단 또는 백업

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    timer?.invalidate()
    timer = nil
}
```

- 사용자가 앱을 잠시 떠날 때 백그라운드에서 실행되는 타이머 중단

---

### ✅ 5. 알림 스케줄링

```swift
func sceneDidEnterBackground(_ scene: UIScene) {
    let content = UNMutableNotificationContent()
    content.title = "앱으로 돌아오세요!"
    content.body = "다시 시작해보세요~"
    let trigger = UNTimeIntervalNotificationTrigger(timeInterval: 60, repeats: false)
    let request = UNNotificationRequest(identifier: "reminder", content: content, trigger: trigger)
    UNUserNotificationCenter.current().add(request, withCompletionHandler: nil)
}
```

- 앱이 백그라운드로 전환되었을 때 푸시 알림 예약
