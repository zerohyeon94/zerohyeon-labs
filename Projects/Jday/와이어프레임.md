## 🏠 홈 화면 (iOS)

**확정안: A안 + C안 숫자 카드 조합**

- 상단: 날짜 + 요일 + 숫자 요약 카드 (할 일 남음 / 오늘 일정 / 진행률)
- 섹션 1: 어제 미완료 카드 + "모두 오늘로 옮기기" 버튼
- 섹션 2: 오늘 할 일 체크리스트 (완료 항목 취소선 처리 + 스와이프 삭제)
- 섹션 3: 오늘 일정 타임라인
- 하단: 탭바 (홈 / 캘린더 / ⚡FAB / 이슈 / 설정)

---

## ⚡ 빠른 추가 바텀시트 (iOS)

- 홈 화면 위로 바텀시트 슬라이드업, 배경 딤 처리
- 상단 탭: 할 일 / 일정 / 이슈 전환
- 할 일 선택 시: 제목 + 날짜(오늘 기본값) + 우선순위(낮음/중간/높음)
- 하단 저장 버튼 고정

---

## 📅 캘린더 화면 (iOS)

- 상단: 월 표시 + 이전/오늘/다음 네비게이션
- 주간/월간 세그먼트 전환 컨트롤
- 월별 캘린더 그리드 (날짜별 이벤트 도트 + 기간 일정 바/칩 표시)
- 선택 날짜 하단: 일정 타임라인 + 할 일 리스트 + 완료 기록 통합

---

## ⚠️ 이슈 목록 (iOS)

- 상단: 미해결 개수 표시
- 필터 탭: 전체 / 미해결 / 해결완료
- 이슈 카드: 제목 + 등록일 + 상태 뱃지 (미해결 파랑 / 해결완료 회색)
- 카드 탭 시 상세 화면으로 이동

---

## ⚠️ 이슈 상세 (iOS)

- 상단: 뒤로가기 + 상태 뱃지
- 이슈 제목 (대형)
- 상세 내용 카드
- 메타 정보 카드: 등록일 / 알림 시간 / 상태
- 하단 고정: "해결완료로 표시" 버튼

---

## 🖥️ macOS 홈

- 왼쪽 사이드바: 홈 / 캘린더 / 이슈 / 설정 메뉴
- 중앙 메인: 숫자 카드 3개 + 어제 미완료 + 오늘 할 일
- 우측 패널: 오늘 일정 타임라인
- 상단 툴바: 검색 + 빠른 추가 버튼 팝오버

---

## 🖥️ macOS 캘린더

- 중앙: 월별 그리드 (날짜 셀 내 일정 칩 표시)
- 우측 패널: 선택일 일정 + 할 일 리스트
- 주간/월간 세그먼트 전환

---

## 🖥️ macOS 이슈

- 왼쪽: 이슈 목록 + 필터 탭 (Mail 앱 패턴)
- 우측: 이슈 상세 패널 + 해결완료 버튼

---

## 🖥️ macOS 설정

- 푸시 알림 토글
- 이메일 요약 토글
- 기본 알림 시간 / 시작 화면 / 한 주 시작 요일
- 완료 항목 자동 숨김 토글
- 프로필 관리 + 로그아웃
