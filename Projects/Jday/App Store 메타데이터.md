## 설명

```text
Jday는 하루 업무를 흐름 있게 관리하기 위한 생산성 앱입니다.

할 일, 일정, 이슈를 각각 다른 앱에서 확인하느라 업무 흐름이 끊겼다면 Jday에서 오늘 해야 할 일을 한곳에 정리해보세요.

출근 직후에는 오늘의 할 일과 일정을 빠르게 확인하고, 업무 중에는 떠오른 할 일이나 이슈를 즉시 기록할 수 있습니다. 완료한 일은 날짜별 기록으로 남겨 하루를 돌아보는 데 활용할 수 있습니다.

주요 기능

- 오늘 할 일, 일정, 진행률을 한눈에 확인
- 어제 끝내지 못한 일을 오늘로 이어서 관리
- 할 일, 일정, 이슈 빠른 추가
- 날짜별 일정과 완료 기록 확인
- 이슈 알림과 해결 상태 관리
- iPhone, iPad, Mac 지원
- iCloud 기반 기기 간 동기화

Jday는 복잡한 프로젝트 관리 도구보다 가볍고, 기본 캘린더나 미리알림보다 업무 흐름에 집중한 개인 업무 관리 앱을 목표로 합니다.

매일 해야 할 일과 놓치면 안 되는 이슈를 한 화면에서 정리하고, 오늘의 흐름을 더 선명하게 관리해보세요.
```

## App Store Connect 입력 체크

- [ ] 앱 이름 `Jday` 입력
- [ ] 부제 `할 일·일정·이슈 관리` 입력
- [ ] 기본 카테고리 `생산성` 선택
- [ ] 보조 카테고리 `비즈니스` 선택
