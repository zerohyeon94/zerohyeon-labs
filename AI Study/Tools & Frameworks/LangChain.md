> [!ABSTRACT] 핵심 요약
>
LLM을 활용한 애플리케이션 개발을 쉽게 만들어주는 표준 프레임워크 (접착제 역할).

#### **1) 정의 (Definition)**

- **LangChain**은 개발자가 LLM(GPT, Claude 등)을 다른 데이터 소스(PDF, 웹, DB)나 계산 도구와 **'사슬(Chain)'처럼 연결**하여 복잡한 애플리케이션을 만들 수 있게 도와주는 오픈소스 라이브러리(Python/JavaScript)입니다.

#### **2) 핵심 구성 요소**

- **Chains:** 여러 작업(프롬프트 입력 -> 모델 실행 -> 결과 가공)을 순차적으로 연결.
    
- **Agents:** 상황에 따라 어떤 행동을 할지 모델이 판단하게 함 (앞서 배운 Agent 개념을 구현).
    
- **Memory:** 대화의 맥락(이전 대화 내용)을 저장하고 불러오는 기능.

#### **3) 비유**

- **LLM**이 '뇌'라면, **LangChain**은 그 뇌에 팔다리(Tools)를 붙이고, 업무 매뉴얼(Chains)을 쥐여주고, 기억 노트(Memory)를 제공하는 **'작업반장'**입니다.
