# Core ML
### 머신러닝 모델을 여러분의 앱에 통합하세요

**지원 환경**

- iOS 11.0+ / iPadOS 11.0+ / tvOS 11.0+
    
- macOS 10.13+ / Mac Catalyst 13.0+
    
- watchOS 4.0+
    
- visionOS 1.0+

## 개요

Core ML을 사용하면 머신러닝 모델을 앱에 손쉽게 통합할 수 있습니다. Core ML은 모든 모델에 대해 통합된 표준 규격을 제공합니다. 여러분의 앱은 Core ML API와 사용자 데이터를 활용하여 예측을 수행하며, 이 모든 과정과 모델의 학습 및 미세 조정(fine-tune)이 사용자의 기기 내부(On-device)에서 이루어집니다.

### 모델의 역할
'모델'이란 학습 데이터 세트에 머신러닝 알고리즘을 적용한 결과물입니다. 이 모델을 통해 새로운 입력 데이터를 바탕으로 예측을 내릴 수 있습니다. 모델은 전통적인 코딩 방식으로 작성하기에는 너무 복잡하거나 불가능한 다양한 작업들을 수행합니다. 예를 들어, 사진의 픽셀 정보를 직접 분석해 사진을 분류하거나 특정 객체를 찾아내도록 모델을 학습시킬 수 있습니다.

### 모델의 생성 및 변환
Xcode에 내장된 **Create ML** 앱을 사용하면 모델을 직접 만들고 학습시킬 수 있습니다. Create ML로 만든 모델은 이미 Core ML 형식이므로 앱에 바로 적용할 수 있습니다. 대안으로, 다른 다양한 머신러닝 라이브러리(예: PyTorch, TensorFlow)로 학습시킨 모델을 **Core ML Tools**를 활용해 Core ML 형식으로 변환하여 사용할 수도 있습니다. 일단 모델이 기기에 탑재되면, 해당 기기 사용자의 데이터를 바탕으로 모델을 재학습시키거나 미세 조정할 수 있습니다.

### 성능 최적화 및 개인정보 보호
Core ML은 메모리 사용량과 배터리 소모를 최소화하는 동시에 기기의 CPU, GPU, 그리고 Neural Engine(신경망 엔진)을 지능적으로 활용하여 On-device 성능을 극대화합니다. 모델이 오직 사용자의 기기 안에서만 실행되기 때문에 네트워크 연결이 전혀 필요 없으며, 이는 사용자의 개인 데이터를 안전하게 보호하고 앱의 반응 속도를 매우 빠르게 유지하는 데 도움을 줍니다.

**프레임워크 구조** Core ML은 특정 분야에 특화된 Apple의 다른 프레임워크들을 떠받치는 기초 뼈대 역할을 합니다.

- **Vision:** 이미지 분석
    
- **Natural Language:** 텍스트 자연어 처리
    
- **Speech:** 음성을 텍스트로 변환
    
- **Sound Analysis:** 오디오 속 소리 식별

Core ML 그 자체는 Metal Performance Shaders(그래픽 처리 가속)뿐만 아니라, Accelerate 및 BNNS와 같은 저수준 기반 요소(Low-level primitives)들 위에서 작동하도록 설계되었습니다.
